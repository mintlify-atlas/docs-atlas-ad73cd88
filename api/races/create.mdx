title: Create race
description: Create a new race record in the database
---

## Endpoint

```
POST /api/races/create
```

Create a new race record in the F1 PitLane Predict database. This endpoint requires admin authentication via JWT token.

## Authentication

This endpoint requires admin privileges. Include a valid JWT token in the request body with `userType` set to `admin`.

## Request body

<ParamField body="token" type="string" required>
  JWT authentication token with admin privileges
</ParamField>

<ParamField body="raceId" type="integer" required>
  Unique identifier for the race
</ParamField>

<ParamField body="season" type="string" required>
  Season year (4-character string)
</ParamField>

<ParamField body="round" type="integer" required>
  Round number in the season
</ParamField>

<ParamField body="raceName" type="string" required>
  Name of the race
</ParamField>

<ParamField body="circuitId" type="string" required>
  Unique identifier for the circuit
</ParamField>

<ParamField body="circuitName" type="string" required>
  Name of the circuit
</ParamField>

<ParamField body="locality" type="string" required>
  City or locality where the race takes place
</ParamField>

<ParamField body="country" type="string" required>
  Country where the race takes place
</ParamField>

<ParamField body="date" type="string" required>
  Race date (ISO 8601 format)
</ParamField>

<ParamField body="time" type="string" required>
  Race time
</ParamField>

<ParamField body="url" type="string" required>
  URL with additional race information
</ParamField>

### Example request

```bash
curl -X POST https://api.f1pitlanepredict.com/api/races/create \
  -H "Content-Type: application/json" \
  -d '{
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "raceId": 1,
    "season": "2024",
    "round": 1,
    "raceName": "Bahrain Grand Prix",
    "circuitId": "bahrain",
    "circuitName": "Bahrain International Circuit",
    "locality": "Sakhir",
    "country": "Bahrain",
    "date": "2024-03-02T00:00:00.000Z",
    "time": "15:00:00",
    "url": "https://en.wikipedia.org/wiki/2024_Bahrain_Grand_Prix"
  }'
```

## Error responses

<ResponseField name="400 Bad Request">
  Missing or invalid token in request body
  
  ```json
  {
    "status": 400,
    "body": "{\"error\": \"Bad Request\"}"
  }
  ```
</ResponseField>

<ResponseField name="401 Unauthorized">
  JWT verification failed or token has expired
  
  ```json
  {
    "status": 401,
    "body": "{\"error\": \"Unauthorized\"}"
  }
  ```
  
  or
  
  ```json
  {
    "status": 401,
    "body": "{\"error\": \"Token has expired\"}"
  }
  ```
</ResponseField>

<ResponseField name="403 Forbidden">
  User does not have admin privileges
  
  ```json
  {
    "status": 403,
    "body": "{\"error\": \"Forbidden\"}"
  }
  ```
</ResponseField>
