## Endpoint

```
GET /api/races
```

Retrieve a list of all races in the F1 PitLane Predict database. This endpoint returns all race records including season information, circuit details, and race dates.

## Request

This endpoint does not require any parameters or authentication.

### Example request

```bash
curl -X GET https://api.f1pitlanepredict.com/api/races
```

## Response

<ResponseField name="data" type="array">
  Array of race objects
  
  <Expandable title="Race object properties">
    <ResponseField name="raceId" type="integer">
      Unique identifier for the race
    </ResponseField>
    
    <ResponseField name="season" type="string">
      Season year (4-character string)
    </ResponseField>
    
    <ResponseField name="round" type="integer">
      Round number in the season
    </ResponseField>
    
    <ResponseField name="raceName" type="string">
      Name of the race
    </ResponseField>
    
    <ResponseField name="circuitId" type="string">
      Unique identifier for the circuit
    </ResponseField>
    
    <ResponseField name="circuitName" type="string">
      Name of the circuit
    </ResponseField>
    
    <ResponseField name="locality" type="string">
      City or locality where the race takes place
    </ResponseField>
    
    <ResponseField name="country" type="string">
      Country where the race takes place
    </ResponseField>
    
    <ResponseField name="date" type="string">
      Race date (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="time" type="string">
      Race time
    </ResponseField>
    
    <ResponseField name="url" type="string">
      URL with additional race information
    </ResponseField>
  </Expandable>
</ResponseField>
