---
title: Delete race
description: Delete a race record from the database
---

## Endpoint

```
POST /api/races/delete
```

<Note>
Despite being a POST request, this endpoint performs a delete operation. It's configured this way in the Nitro.js handler.
</Note>

Delete a race record from the F1 PitLane Predict database. This endpoint requires admin authentication via JWT token.

## Authentication

This endpoint requires admin privileges. Include a valid JWT token in the request body with `userType` set to `admin`.

## Request body

<ParamField body="token" type="string" required>
  JWT authentication token with admin privileges
</ParamField>

<ParamField body="raceId" type="integer" required>
  Unique identifier of the race to delete
</ParamField>

### Example request

```bash
curl -X POST https://api.f1pitlanepredict.com/api/races/delete \
  -H "Content-Type: application/json" \
  -d '{
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "raceId": 1
  }'
```

## Response

<ResponseField name="status" type="integer">
  HTTP status code
</ResponseField>

<ResponseField name="body" type="string">
  JSON stringified response containing the deleted race object or error message
  
  <Expandable title="Success response properties">
    <ResponseField name="raceId" type="integer">
      Unique identifier for the deleted race
    </ResponseField>
    
    <ResponseField name="season" type="string">
      Season year (4-character string)
    </ResponseField>
    
    <ResponseField name="round" type="integer">
      Round number in the season
    </ResponseField>
    
    <ResponseField name="raceName" type="string">
      Name of the race
    </ResponseField>
    
    <ResponseField name="circuitId" type="string">
      Unique identifier for the circuit
    </ResponseField>
    
    <ResponseField name="circuitName" type="string">
      Name of the circuit
    </ResponseField>
    
    <ResponseField name="locality" type="string">
      City or locality where the race takes place
    </ResponseField>
    
    <ResponseField name="country" type="string">
      Country where the race takes place
    </ResponseField>
    
    <ResponseField name="date" type="string">
      Race date (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="time" type="string">
      Race time
    </ResponseField>
    
    <ResponseField name="url" type="string">
      URL with additional race information
    </ResponseField>
  </Expandable>
</ResponseField>

```json
{
  "status": 200,
  "body": "{
    \"raceId\": 1,
    \"season\": \"2024\",
    \"round\": 1,
    \"raceName\": \"Bahrain Grand Prix\",
    \"circuitId\": \"bahrain\",
    \"circuitName\": \"Bahrain International Circuit\",
    \"locality\": \"Sakhir\",
    \"country\": \"Bahrain\",

    \"time\": \"15:00:00\",
    \"url\": \"https://en.wikipedia.org/wiki/2024_Bahrain_Grand_Prix\"
  }"
}
```

## Error responses

<ResponseField name="400 Bad Request">
  Missing or invalid token in request body
  
  ```json
  {
    "status": 400,
    "body": "{\"error\": \"Bad Request\"}"
  }
  ```
</ResponseField>

<ResponseField name="401 Unauthorized">
  JWT verification failed or token has expired
  
  ```json
  {
    "status": 401,
    "body": "{\"error\": \"Unauthorized\"}"
  }
  ```
  
  or
  
  ```json
  {
    "status": 401,
    "body": "{\"error\": \"Token has expired\"}"
  }
  ```
</ResponseField>

<ResponseField name="403 Forbidden">
  User does not have admin privileges
  
  ```json
  {
    "status": 403,
    "body": "{\"error\": \"Forbidden\"}"
  }
  ```
</ResponseField>
